# MediCareIQ

MediCareIQ is a digital healthcare platform in Nepal that offers video consultations, a single-payment system, machine learning algorithms, and access to labs, enhancing diagnostic precision and empowering patients for proactive health management.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. The project is divided into three main parts: the Frontend, the Backend, and the `appointment_fast` service.

## Prerequisites

Before you start, ensure you have installed the following:
- Node.js
- Yarn
- Python
- pip
- uvicorn

uvicorn app.main:app --reload
You can check if they're installed by running:
```bash
node --version
yarn --version
python --version
pip --version
uvicorn --version
```
# Installing

## Frontend 
```
$ yarn install
$ yarn run test
```

## Backend 

```
npm i
npm start
```


## appointment_fast

```
pip install -r requirements.txt
uvicorn app.main:app --reload
```
